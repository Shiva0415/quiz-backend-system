# Quiz Backend System

## 🚀 Overview

A backend system for managing quizzes, questions, and user attempts built using Spring Boot.

## 🛠 Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL

## ✨ Features

* Create and manage quizzes
* Add and retrieve questions
* Layered architecture (Controller-Service-Repository)
* Global exception handling

## 📂 Project Structure

src/
├── controller/
├── service/
├── repository/
├── entity/
├── exception/

## ▶️ How to Run

1. Clone the repository
2. Configure MySQL in `application.properties`
3. Run using:
   mvn spring-boot:run
